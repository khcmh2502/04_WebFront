DOM(Document Object Model)

    HTML(웹문서, 웹페이지) 문서를 객체 기반으로 표현한 것
    -> HTML 문서에 작성된 내용을 트리구조(계층형으로 표현)로 나타냈을 때
        각각의 태그, TEXT, COMMENT 등을 Node라고 한다.

    document : { 
        DOCTYPE : html,
        HTML : {
            HEAD : {
                TITLE : { TEXT : "문서 제목" },
                STYLE : {...},
                META : {...}
            } ,
            BODY : {
                H1 : { TEXT : "제목", ATTRIBUTE : "속성" },
                COMMENT : {TEXT : "주석 내용"},
                DIV : {...}
                ul#test{ childNodes : [], innerText : 123,   }
            }
        }
    }
